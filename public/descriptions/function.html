Function is a protocol which mints FBTC on various blockchains.

FBTC exists in two forms: Standard FBTC and Locked FBTC (FBTC1), where FBTC1 also generates yield.

Standard FBTC is minted on several blockchains:

  • Ethereum
  • Mantle
  • Bob
  • BSC
  • Arbitrum
  • Base
  • Berachain
  • Sonic

Locked FBTC is minted on several blockchains:

  • Ethereum
  • Bob
  • BSC
  • Sonic
  • Mantle
  • Arbitrum
