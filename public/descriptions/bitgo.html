BitGo is a custodian which mints WBTC on various blockchains.

Canonical WBTC is minted on several blockchains:

  • Ethereum
  • Solana
  • Base
  • Kava
  • Osmosis

All the other WBTC are bridged versions (mostly from Ethereum).

WBTC can be minted and burned only by a set of whitelisted merchants. One can see the list of merchants here.
