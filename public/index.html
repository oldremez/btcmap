Network Frames

Nodes are grouped by blockchain networks, each with its own color and border. The connections between nodes show how Bitcoin moves between different blockchains and protocols.

Node links

The connection from one node to another means one of the who:

  • Issuer -> Token link means that the token is minted by an issuer
  • Token -> Issuer link means that the token is used as collateral/backing by the issuer (usually issuer is a wrapper or bridge)
  • Token -> Protocol link means that the token is locked in a prototol

The number on the link represents the number of tokens minted/locked

When it's possible, the numbers are queried on-chain. However, sometimes it's tough to get the numbers from the chain (especially when it comes to BTC reserves: there are usually a lot of addresses to query and it's almost impossible to track them all), so some numbers are queried from the third party API. See source code for details.
